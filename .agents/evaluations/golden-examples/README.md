# Golden Examples

This directory stores exemplary outputs that represent the quality bar
agents should aim for.

## What Belongs Here

- High-quality agent output examples
- Well-structured feature briefs
- Excellent code review reports
- Clear architecture decision records
- Well-formatted transcription summaries

## Purpose

Golden examples serve as:

1. **Quality benchmarks** — show agents what "good" looks like.
2. **Training data** — provide few-shot examples for prompts.
3. **Regression checks** — compare current output against known-good output.
4. **Onboarding material** — help new users understand expected quality.

## Naming

```
feature-brief-golden.md
code-review-golden.md
architecture-decision-golden.md
transcription-summary-golden.md
```

## Guidelines

- Only add outputs that meet score 4+ on the relevant rubric.
- Include the rubric scores with each golden example.
- Update golden examples when standards evolve.
